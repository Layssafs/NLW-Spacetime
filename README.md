## 💻 Projeto
Esse é um projeto web responsivo de uma cápsula do tempo para exibir memórias em uma linha do tempo.

## 🚀 Tecnologias
Esse projeto foi desenvolvido durante a NLW data Rocketseat com as seguintes tecnologias:

-HTML
-CSS
-Git e Github
